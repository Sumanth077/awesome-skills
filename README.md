> 10,000+ web scraping tools for Claude Code, Cursor, Codex & Gemini CLI

Build agent skills that extract data from any platform. Connect AI coding assistants to live web data through Apify Actors.

## ⚡ Quick Start
```bash
# Add the marketplace
/plugin marketplace add https://github.com/apify/agent-skills

# Install a skill
/plugin install apify-ultimate-scraper@apify-agent-skills
```
